# 🌍 Language Learning App

An interactive and beginner-friendly Language Learning App built using **HTML, CSS, and JavaScript**. The application helps users learn new words and practice through quizzes with a clean and responsive interface.

## 🚀 Features

* 📚 Vocabulary flashcards
* 🔊 Pronunciation using Speech Synthesis API
* 📝 Interactive quiz section
* 🎯 Score tracking
* 💾 Local storage support for saving progress
* 📂 Category-based learning
* 📱 Responsive and user-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Browser Local Storage
* Speech Synthesis API

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Language-Learning-App.git
```

2. Open the project folder in Visual Studio Code.

3. Install the **Live Server** extension.

4. Right-click on `index.html` and select:

```
Open with Live Server
```

## 🎮 How to Use

1. Select a category.
2. View vocabulary flashcards.
3. Click **Pronounce** to hear the word.
4. Use **Next Word** to browse more words.
5. Take quizzes and improve your score.
6. Progress is automatically saved using Local Storage.
